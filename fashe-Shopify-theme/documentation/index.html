Overview

Looking for a great ecommerce store to sell your items is your necessity now? Don’t hesitate, Sophie Responsive Shopify theme is not a bad choice for you!

Sophie is a Shopify theme. From headers and menus to extensions, Sophies is flexible. Header is small and neat with logo, icons of topcart, setting and search. Slide is smooth with transitions of text and images. Most Viewed Products Slider with “Load More” button which show more many top viewed products. Category Tabs Slider show products of categories with number of products of each category. Daily deals module show timer and description of special products. Effect when hover on each product is very smooth with Add To Cart, Add To Wishlist, Add To Compare button. “New/Sale” labels are very nice and highlight in new and special products. Static blocks with elegant images and useful text, nice font bring fresh feelings for this theme. Posts are in Blog with advertisment content. Brand Logo Slider displays favorite logo and related logo in slider. Easy to custom these modules in Admin panel. When hover on a product, other side of that product will be shown via Product Rotator Image feature. Footer is also neat with introduction about your store, links, paypal icon, copyright.

Don’t miss Sophie responsive Shopify theme!

Theme Installation

Follow the following instructions to install the theme on your store. Note that most of the images are from one of our other shopify theme but the process is simply the same

  1. From your shop dashboard, go to Online Store > Themes and click on upload theme link. 
  2. Select the zip version of the theme and click on Upload button. 
  3. Once uploaded you'll see the theme listing below the published theme. Click on the publish button to make it live.

Demos Setup

Theme comes with multiple demos. Follow the following instructions to setup the store theme based on the demos. 

Sample images

Sample images has been added to the zip with the name 'demo-images' in the folder 'demos-data' in case you want to setup the store with demo data. Follow the following steps to upload demo images.

  1. Go to the settings > Files and click on upload button as can be seen in the screenshot. 
  2. Select all the files in demos-data/demo-images folder and upload to server.

Sample demo config files

Sample config json fiiles has been added to the zip with the name 'settings-data' in the folder 'demos-data' in case you want to setup the store with any of the four demos. Follow the following steps to upload demo settings files to your store.

  1. Go to the Online Store > Themes > Edit Html/CSS as can be seen in the screenshot.
     
  2. There are four config files in demos-data/settings-data folder of the zip.Open the files based on the demo that you want to setup.
  3. Select all the content of the file and upload it to the settings_data.json file of the theme located in config folder as can be seen in the screenshot 
  4. Open the store url and theme will be setup according to your demo.

Now you can also add videos against your products. All you need to do is to add a video link inside a video tag at the bottom of your product page after all the content. It should be like

Section Slider

This is the first section that is added to most of the themes. You'll see it already added with demo. 

There a lots of options that you can customize on the page

  1. First option to either set/unset auto rotation of slides.
  2. Second field allow you to set time of each slide for display
  3. Third option allows you to select the slider design to show

Next thing is to add content. You can add two types of content. A slider image which is shown on slider and side image which are shown on some designs on the side of sliders.

A maximum of 6 sliders can be added to each slideshow. For each slider there is

  1. Image upload to background image. Leave blank if you don't want to have background image.
  2. Text fields to add title and description text.
  3. Button link fields to add link text and link

Contact Page

In order to use Contact Us page first you need to create a page by going to

Add title and select page.content as template. 

The contact information are added from footer contact information. For map you need to go to google.com/maps for generating code and add that code onto theme settings > contact page settings area. 

 

Collection Page

Setup Multiple Currencies

1. Go to Admin > Settings > General

2. Scroll down to Standards & formats area, you'll see your Currency setting.

3. Click change formatting to open the Currency Formatting dialog.

4. Change HTML with currency and HTML without currency elements in the dialog.
